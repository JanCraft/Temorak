**[World-File-Definition]**

Worlds are JSON files defined as follows:
- Version *(TDR)*
- Tiles:
    **ID, *(String)***
    Texture Name, *(String)*
    **X, *(Integer)***
    Y, *(Integer)*
    **Metadata *(optional, just a JSON dictionay)***
- Entites:
    **ID, *(String)***
    Texture Name, *(String)*
    **X, *(Integer)***
    Y, *(Integer)*
    **Metadata *(optional, just a JSON dictonary)***
- World Metadata *(Contains, for example, regions)*
